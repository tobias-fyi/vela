Returns `True` if the given number is even, `False` otherwise.

Checks whether a number is odd or even using the modulo (`%`) operator. 
Returns `True` if the number is even, `False` if the number is odd.

```py
def is_even(num):
  return num % 2 == 0
```

```py
is_even(3) # False
```
